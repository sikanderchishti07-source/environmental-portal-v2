Quick Guide

  1. 1. Paste your Admin API Key in the sidebar before doing anything.
  2. 2. Create a New Client — the system auto-generates a permanent unique ID (e.g. DIR-SAS-001). Give this ID to the client.
  3. 3. Upload Reports linked to that Client ID, 3–4 days after site work.
  4. 4. Set isPublished = true to make the report visible in the client portal.
  5. 5. Clients enter their ID on your website to see all their reports.
  6. 6. Check Inspection Requests to view and manage booking submissions.
